# Full Stack Open

Este repositório contém as soluções para os exercícios do curso **Full Stack Open** da Universidade de Helsinque.

## Estrutura do Projeto

O projeto está dividido em partes, cada uma contendo os exercícios correspondentes:

- **part1**: Introdução ao React e JavaScript
- **part2**: Componentes e Props
- **part3**: Gerenciamento de Estado e Eventos
- **part4**: Hooks e Componentes de Ordem Superior
- **part5**: Comunicação com o Servidor
- **part6**: Formulários e Validação
- **part7**: Roteamento e Estilização
- **part8**: Testes e Integração Contínua
- **part9**: Banco de Dados e Autenticação
- **part10**: Deploy e CI/CD

## Como Executar

Cada parte pode ser executada de forma independente. Siga os passos abaixo:

1. Navegue até a pasta da parte desejada:
   ```bash
   cd part1
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm start
   ```
